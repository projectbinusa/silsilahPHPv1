D) Structure - top

The structure of the files of Puerto Family Tree Script, is very simple:

  1. In the "root" there are the PHP pages of Puerto Family Tree Script.
  2. In the folder "configs/" and subfolders there are the files of the Puerto Family Tree Script system
  3. In the folder "css/" there are CSS files of the Puerto Family Tree Script system
  4. In the folder "documentation/" there is the documentation page.
  5. In the folder "img/" there are Images of the Puerto Family Tree Script system
  6. In the folder "js/" there are Javascript files of the Puerto Family Tree Script system
  7. In the folder "sass/" there are Sass files of the Puerto Family Tree Script for the developpers
  8. In the folder "upload/" there are the members images that uploaded in your server.
  9. In the folder "webfonts/" there is the script's fonts.

E) PHP Structure - top

To simplify the reading, we have divided the structure into two parts:

  1. PHP pages (in the "root"):
    2. - ajax.php, this script displays queries of system.
    - details.php this script displays the family tree details for edit.
    - footer.php, this script displays a preview of a footer.
    - header.php, this script displays a preview of a header page.
    - index.php, this script displays a preview of index page contain the login & register form.
    - intall.php, this script displays the installation file.
    - list.php, this script displays the public families list.
    - mnewmember.php, this script displays a preview of the modal that contain the add new member feature.
    - tree.php, this script displays a preview of the main family tree page.

  2. PHP system files (in the "configs/" folder):
    3. - class.upload.php, this script displays the files upload class of the system.
    - config.php, this script performs the database operations.
    - connection.php, this script displays the connections info of the system.
    - function.all.php, this script displays the functions of the system.
    - lang/, the main languages of systeme.

F) CSS Files and Structure - top

I'm using these CSS files in this item.

bootstrap.min.css this css file contain the Bootstrap v4.4.1 elements.

all.min.css this css file contain all Font Awesome icons.

jquery-confirm.min.css this css file is responsible for the jquery jConfirm Plugin.

style.css contains all of the specific stylings for the Puerto Family Tree Script.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.
